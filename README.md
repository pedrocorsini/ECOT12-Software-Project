## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/thor-verse.git
   cd thor-verse
   ```

2. Build the project using Maven:
   ```bash
   mvn clean install
   ```

3. Run the application (example):
   ```bash
   mvn exec:java -Dexec.mainClass="br.eng.corsini.thor.Main"
   ```
